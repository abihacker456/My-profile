Video Projector 2 - Technical Details

Challenge: The app initially failed to build due to Java 24 and Gradle 8.13 incompatibility. The error message "Launcher JVM: 24.0.1" indicated a version mismatch that prevented compilation.

Solution: Successfully upgraded the project to Gradle 8.14.1, configured the JDK toolchain for Java 17 compatibility, and resolved dependency synchronization issues. This required manual cache cleaning and Gradle wrapper updates.

Technologies: Kotlin, Android SDK, Gradle, ViewBinding, MediaPlayer API, CameraX, File System API.

Key Features:

  • Video playback for MP4, AVI, MKV formats with gesture controls
  • Image projection with pinch-to-zoom and rotation
  • Integrated file browser for local storage access
  • Camera projection mode for real-time display
  • Professional About screen with developer information
